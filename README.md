# ☕ Coffee Shop Business Analysis

## Project Overview
Analysis of 149,116 real transactions from a NYC coffee shop chain 
with 3 locations to extract business insights using Python.

## Business Questions Answered
- Which product category generates the most revenue?
- What are the peak hours of the day?
- Which store location performs best?

## Key Findings
- ☕ Coffee is #1 — $270,000 revenue (36% of total)
- 🕐 Peak hours: 8AM–10AM (highest footfall)
- 📍 All 3 stores equally profitable (~33% each)
- ❌ Packaged Chocolate — lowest revenue, consider discontinuing

## Tools Used
- Python, Pandas, Matplotlib
- Google Colab
- Dataset: 149,116 transactions | 11 columns

- `Coffee_Shop_Analysis.ipynb` — Full analysis code
